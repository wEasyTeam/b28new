使用步骤说明：

1、将要提取文字的 工程文件夹/文件 拷贝到此目录下
2、请在控制台（cmd）定位到textPicker文件夹，然后执行：
	node node_b28.js.
3、输入要提取的文件夹或者文件名称，如果需要指定输出文件的名称，请追加输入：/输出文件的名称
	即输入格式：FolderName|FileName[/OutPutFileName]
4、自行检查和整理过滤提取好的文件。

warning：
1、目前为止，此工具支持[ html文件中DOM部分的文本节点] && [JS里的中文字符串] 提取，
2、如果js文件中存在作为html插入的字符串 会出现提取错误，请自行修正。
